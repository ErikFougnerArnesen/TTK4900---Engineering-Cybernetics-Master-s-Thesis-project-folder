# TTK4900---Engineering-Cybernetics-Master-s-Thesis-project-folder
MATLAB code repository for a master’s thesis on guidance and control of a cylinder-shaped agile vehicle. Developed in MATLAB R2025a and requiring the MSS Toolbox.


This repository contains the MATLAB code used in the development, implementation, and evaluation of the guidance and control system for my master’s thesis on a cylinder-shaped agile vehicle. The project includes simulation files, vehicle model implementation OpenVSP model, guidance laws, control system design, test scenarios, and scripts used to evaluate flight performance.

The simulation framework was developed in MATLAB R2025a and requires the MSS Toolbox to run. The code was created as part of a complete simulation-based design process, where aerodynamic data, vehicle dynamics, guidance algorithms, and control strategies were combined to investigate the behavior and performance of the vehicle in flight.

MSS Toolbox can be found in: https://github.com/cybergalactic/MSS Developed by Professor Thor Inge Fossen

## Components
* Simulation Script with waypoints and user GUI for selecting Control strategy and wind conditions
* Implementation of dryden gust wind model
* Implemented control stategies 
* Vehicle model parameters
* Vehicle Equation of Motion
